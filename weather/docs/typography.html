Typography

The title banner of the page is using the Anton sans-serif typography. This is done in order to create a more friendly banner, that is inviting with the logo.

The rest of the fonts are Montserrat Sans-serif. These are used in order to create uniformity and make reading and navigating more easy throughout the page.
